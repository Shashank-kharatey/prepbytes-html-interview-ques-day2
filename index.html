Q.1 Difference between HTML tags and elements?

Ans. An HTML 'element' is an individual component of an html document that holds an existing meaning in its own. Technically html element is the collection of starts tag, its attributes(if needed), content to be displayed or work upon(if any), and the closing/end tag.
Whereas
HTML tag (either opening or closing) is used to work to mark the start or end of an element, they doesn't poses any meaning by their own unless used with complete valid syntax and finally turning into element.

Q.2 Explain br tag?

Ans. 'br' tag is basically 'The line break element', it is used in the web page to provide a line break/starting annew line to the content in which it is used. It is a self closing tag by nature.

Q.3 Explain how many different types of lists we have in HTML?

There are 3 types of 'Lists' we have in HTML, they are 'ordered lists','unordered lists','description lists'.

1. ordered list- This list works with different types of list marking format such as(123,i ii iii,I II III,abc

2. unordered list- This list works for marking the lists with different types of bullet forms such as (circle,square,none,etc)

Description list- This list is used when the content/item of the list has further description, it is also known as definiton list.

Q.4. What are semantic tags?

'Semantic tags' as per the meaning of "semantic"(study related to meaning of the words) are used in HTML for better describing the meaning/work function to the browser and the developer. for eg. table tag (tells about the data format is in table form)
